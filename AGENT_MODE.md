# 智能体模式 (Agent Mode)

从 v0.2.0 开始，QQ 机器人支持**智能体模式**，具备以下能力：

## 功能特性

### 1. 混合记忆系统

| 层级 | 存储内容 | 技术实现 | 用途 |
|------|----------|----------|------|
| **短期记忆** | 当前会话对话历史 | JSON 文件 | 上下文理解 |
| **长期记忆** | 重要对话片段、经验 | ChromaDB 向量库 | 语义检索 |
| **关键事实** | 用户偏好、任务、承诺 | SQLite 数据库 | 结构化查询 |

### 2. 意图识别

自动分析用户消息，分类为：
- **chat** - 闲聊对话
- **task** - 任务请求（需要执行动作）
- **query** - 查询请求（获取信息）
- **command** - 系统命令

### 3. 工具集

内置工具：
- `read_file` / `write_file` - 文件读写
- `list_files` / `search_files` - 文件浏览/搜索
- `run_python` / `run_shell` - 代码/命令执行
- `web_search` / `web_fetch` - 网络搜索/抓取（需配置）

### 4. 任务规划

复杂任务自动分解为子任务，支持：
- 依赖管理
- 优先级排序
- 进度跟踪

### 5. 监督日志

详细记录智能体决策过程：
- 日志文件：`data/logs/supervision_YYYY-MM-DD.jsonl`
- 控制台实时输出
- 支持按级别/模块筛选

## 使用示例

### 1. 带上下文的对话

```
用户：我住在北京，是一名程序员
机器人：好的，我记住了！

用户：我之前是做什么的？
机器人：根据我的记忆，你是一名程序员。
```

### 2. 任务执行

```
用户：帮我分析一下这个项目的文件结构
机器人：我收到任务：分析项目文件结构

我计划分 3 步完成：
1. 列出项目根目录内容
2. 识别主要文件和文件夹
3. 生成结构报告

请确认我开始执行。
```

### 3. 查询记忆

```
用户：我之前说过喜欢什么编程语言？
机器人：根据我的记忆，你提到过喜欢 Python，因为它简洁易用。
```

## 监督日志格式

每条日志包含：

```json
{
  "timestamp": 1714636800.0,
  "level": "decision",
  "module": "IntentRecognizer",
  "message": "识别意图：task (置信度：0.85)",
  "details": {
    "message": "帮我分析一下这个项目",
    "payload": {}
  }
}
```

日志级别：
- `info` - 一般信息
- `decision` - 重要决策
- `action` - 执行动作
- `warning` - 警告
- `error` - 错误

## 注意事项

### 1. 性能考虑

- ChromaDB 首次初始化较慢（约 2-3 秒）
- 向量检索建议 limit <= 10
- 监督日志每 10 条自动刷新一次

### 2. 安全建议

- 文件操作限制在项目目录内
- 代码执行使用沙箱
- 敏感操作需要用户确认

### 3. 故障排查

**问题：ChromaDB 加载失败**

解决：检查是否正确安装
```bash
pip install chromadb>=0.4.0
```

**问题：记忆检索不准确**

解决：增加重要性评分阈值，或手动标记重要对话

**问题：任务执行卡住**

解决：查看监督日志，确认是否有依赖未满足

## 切换回简单模式

如果智能体模式不稳定，可以随时切换回简单模式：

```python
# dialogue.py
AGENT_MODE = False  # 改回 False
```

简单模式保持向后兼容，所有原有功能正常工作。
